Download the Wonkydoodles Dataset!

Download

The Dataset is saved as an SQLite-Database with the following tables and respective structures:

Doodle
ID Int
Category String
Recognized Bool
Timestamp String
Countrycode Two chars
Base64 base64 String
Stroke
ID Int
Doodle_ID Int
Vector
ID Int
Stroke_ID Int
x Int
y Int
t Int

Basically, a Vector is a Child of a Stroke, which in turn is a Child of a Doodle. That way, for every Doodle, you can iterate through its Strokes and the Vectors associated to this Stroke to, for example, draw the Doodle or feed it to an RNN.

The Wonkydoodles Dataset is more or less consistent with he "Quick, Draw!" Dataset

The Wonkydoodles Database is available under the Creative Commons Attribution 4.0 International license, which in plain english means that you can use this for any, even commercial, purpose. Just shout me out when you do use it.
